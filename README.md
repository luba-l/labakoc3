
# Лабораторная работа  

## Параллельная обработка текстовых данных (Producer – Consumer)

## Описание работы

В данной лабораторной работе реализовано приложение на языке **C++**, демонстрирующее
алгоритм параллельной обработки текстовых данных с использованием шаблона  
**Producer → Consumer**.

Программа обрабатывает входной файл с записями пользователей, распределяет обработку
между несколькими потоками и сохраняет результат в выходные файлы.

---

## Формат входного файла

Каждая строка входного файла имеет формат:

Фамилия;Имя;Отчество;НомерТелефона

Пример:

Иванов;Иван;Иванович;89001234501
Сидорова;Елена;Олеговна;+79001234506

---

## Алгоритм работы программы

1. **Producer (производитель)**  
   - читает входной файл построчно;
   - преобразует строки в структуру `Record`;
   - помещает записи во входную блокирующую очередь.

2. **Consumer (потребители)**  
   - работают в нескольких потоках;
   - получают записи из очереди;
   - выполняют проверку корректности данных;
   - имитируют обработку записи;
   - передают корректные записи в выходную очередь.

3. **Writer (записывающий поток)**  
   - получает обработанные записи;
   - группирует их по первой букве фамилии;
   - сохраняет данные в отдельные файлы.

---

## Проверка корректности данных

Для каждой записи выполняются следующие проверки:

- ФИО:
  - допускается только кириллица;
  - запрещены цифры и латиница;
  - допускаются пробелы и дефисы.

- номер телефона:
  - формат `+7XXXXXXXXXX`;
  - формат `8XXXXXXXXXX`.

Некорректные записи пропускаются и выводятся в консоль.

---
